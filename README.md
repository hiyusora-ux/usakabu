# AI Stock Radar

米国株の **半導体 / AI / フィジカルAI** セクターから優良株を総合スコアで抽出し、
毎日のスクリーニング更新と毎時のトピックス（ニュース）をダッシュボードに表示する Web アプリ。

データソースは **yfinance（無料）**。コスト0で動きます。

---

## 起動方法

PowerShell で：

```powershell
cd "C:\Users\Sora Amano\ai-stock-radar"
.\run.ps1
```

初回は仮想環境の作成と依存インストールが走ります（数分）。
起動後、ブラウザで **http://127.0.0.1:8000** を開いてください。

> 初回データ取得（約45銘柄）に1〜2分かかります。表が空のうちは少し待つか「更新」ボタンを押してください。

## 機能

| 機能 | 内容 |
|------|------|
| 優良株ランキング | 成長性・収益性・モメンタム・アナリスト評価を総合スコア化(0-100)して順位付け |
| テーマ切替 | すべて / 半導体 / AI / フィジカルAI |
| トピックス | 各銘柄の最新ニュースを集約、新着順 |
| 自動更新 | トピックス=米東部 取引時間中 毎時 / スクリーニング=毎営業日17:00 ET（引け後） |
| 手動更新 | 画面右上「更新」ボタン |

## スコアリングの中身（`app/screener.py`）

ユニバース内で各指標をパーセンタイル順位化し、重み付け合算：

| 指標 | 重み | 方向 |
|------|------|------|
| 売上成長率 | 20% | 高いほど良い |
| 利益成長率 | 12% | 高いほど良い |
| 3ヶ月モメンタム | 12% | 高いほど良い |
| 粗利率 | 10% | 高いほど良い |
| 純利益率 | 10% | 高いほど良い |
| ROE | 10% | 高いほど良い |
| 6ヶ月モメンタム | 10% | 高いほど良い |
| 目標株価への上昇余地 | 10% | 高いほど良い |
| 予想PER | 6% | 低いほど良い（割安） |

重みは `METRICS` を編集すれば調整できます。

## 銘柄ユニバースの編集

## ディレクトリ構成

```
ai-stock-radar/
├─ app/
│  ├─ main.py        FastAPI 本体・スケジューラ・APIルート
│  ├─ universe.py    テーマ別ユニバース
│  ├─ data.py        yfinance 取得 + JSONキャッシュ
│  ├─ screener.py    スコアリング/ランキング
│  └─ static/        ダッシュボード(HTML/CSS/JS)
├─ data/             生成キャッシュ(screener.json / topics.json)
├─ requirements.txt
├─ run.ps1
└─ README.md
```

## API（参考）

- `GET /api/screener?theme=semiconductor` … ランキング
- `GET /api/topics?theme=ai` … トピックス
- `GET /api/status` … 更新時刻・件数
- `POST /api/refresh` … 手動更新
